Seletores de tag

Bem-Vindo

No CSS nós possuimos o seletores por ID

  • Seletor por tipo: Buscca elementos por um tag HTML específica
  • Seletor por ID (#): Busca elementos através do atributo "id"
  • Seletor de class (.): Busca elementos através do atributo "class"
  • Seletor Universal: Seleciona todos os elementos HTML
